# Exam Study Tool

Final exam prep for Kuratko-Entrepreneurship: Theory, Process, Practice (12e), Chapters 8, 9, 10, and 12.

## Deployment Options

### Vercel (Recommended — free, instant)

1. Push to GitHub
2. Go to [vercel.com](https://vercel.com) → Import repo → Deploy
3. No environment variables needed

### Netlify

1. Connect repo on [netlify.com](https://netlify.com)
2. Build command: `npm run build`
3. Publish directory: `dist`
4. No environment variables needed
